Объявления:
Создание в навчальном Jira таска с названием «Ім'я Прізвище ДЗ 18»
Зарегистрируйтесь на GitHub
Установите сбор GitHub Desktop
Локально клонируйте собственный репозиторий
Для подтверждения терминала в репозитории создайте:
Папку «Имя_Фамилия_HW18»
В созданных папках создать файл «script.sh»
В створенном файле запишите команду для дій які ви конуватимете нижче.
В папке «Name_Lastname_HW18» создать папку «Need_to_check»
В папке «Need_to_check» создайте файл «README.txt» с текстом «Прочитай мене!»
В папке «Need_to_check» создайте три пустых файла «Text1.txt», «Text2.txt» и «Text3.txt»
В папке «Имя_Фамилия_HW18» создать папку «Проверено»
Скопируйте файл «Text1.txt» в папку «Проверено»
Переместите файл «Text2.txt» в папку «Проверено»
Показать файл «Text3.txt» в папке «Need_to_check»
Зробить файл «Script.sh» выконнованим
Создать в GitHub Desktop комиссию с вашими именами
Запустите ваш коміт у віддалений репозиторій
Переведите вашу задачу в колонку «На тестировании» и перекиньте его на мене.
Додайте посылка на вашу задачу в LMS.
Додатково:
Выконайте ДЗ из LMS
